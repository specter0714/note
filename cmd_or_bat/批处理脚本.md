# bat脚本基本语法

## 注释

```bash
:: 这是一个注释
REM 这也是一个注释
```

## 变量赋值与取值

```bash
set a=John
echo %a%
```

## 输入、输出与暂停

**pause**命令用于赞同批处理文件的执行,并提示用户按任意键继续。

```bash
REM /p 是读取的信息，等号后面可以直接写提示信息，会打印出来
set /p a=please_input_a_number
REM echo 后面可以直接接要输出的文本，然后再指定位置%a%，就相当于printf("----%d----", a);
echo hello,%a%
pause
```
